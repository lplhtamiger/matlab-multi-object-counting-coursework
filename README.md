#基于 MATLAB 的多目标物品自动计数与位置标注系统
机器视觉与图像处理课程大作业 - 方向2：多目标物品自动计数与位置标注系统  
##1. 项目简介

本项目基于 MATLAB 和 Image Processing Toolbox 开发，实现对硬币、瓶盖、橡皮、文具或小零件等多目标图像的自动计数与位置标注。系统采用传统数字图像处理流程，包括灰度化、中值滤波、自适应二值化、形态学处理和连通域分析，最终输出每个目标的编号、中心坐标、面积和外接矩形。

##2. 项目功能

- 图像读取与 GUI 可视化显示；
- 原图、灰度图、去噪图、二值化图、形态学处理图和识别结果图 2 行 3 列展示；
- 自动统计目标数量；
- 绘制目标外接矩形、中心点和编号；
- 输出每个目标的中心坐标和面积；
- 支持保存最终识别结果图；
- 支持一键清空界面。

##5. 运行步骤

1. 打开 MATLAB。
2. 将当前工作路径切换到 `code` 文件夹。
3. 在命令行窗口运行：multi_object_count_gui
4. 点击“加载图片”，选择 `images` 中的测试图或自行拍摄的物品图像。
5. 点击“开始识别”，查看处理过程与目标坐标输出。
6. 点击“保存结果”，保存最终识别图。

##6. 算法流程

输入图像→灰度化→中值滤波去噪→自适应二值化→形态学处理：去小区域、闭运算、孔洞填充→连通域分析：面积、质心、外接矩形→目标计数、坐标输出和结果标注

##7. 实验结果

系统能够在背景较为干净、目标边界较明显的图像中完成多目标计数和位置标注。对于硬币、瓶盖等灰度差异明显的目标，二值化与连通域分析效果较稳定。对于浅色橡皮等与背景灰度接近的目标，二值化图中可能出现边缘不完全闭合现象，但最终仍可以通过外接矩形完成基本定位。

##8. 实验结论

本项目完成了多目标图像自动计数与位置标注系统的基本设计。系统界面完整、操作简单、结果显示直观，能够满足课程大作业对源代码、实验素材、项目报告和 README 文档的归档要求。实验也说明，传统图像处理方法具有实现简单、运行速度快、可解释性强的优势，但对光照、背景复杂度和目标颜色差异较敏感，后续可结合光照补偿或深度学习方法进一步提升鲁棒性。
